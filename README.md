## Setup

Note : Required Go version >= go1.20.0

- Check go version

```bash
go version
```

- Install dependencies

```bash
go mod tidy
or
go get -u
or
go mod download
or
go install
```

- Copy .env.example to .env

```bash
cp .env.example .env
```

- Run Web Server

```bash
make run
```
